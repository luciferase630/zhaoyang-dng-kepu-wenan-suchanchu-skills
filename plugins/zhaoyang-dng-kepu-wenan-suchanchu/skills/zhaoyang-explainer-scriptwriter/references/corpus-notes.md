# Corpus Notes

Use these notes only as style provenance and boundary guidance. Do not quote or copy old scripts unless the user explicitly asks to work on a provided source file.

## Style Source Boundaries

The skill was abstracted from completed public-education materials where a spoken script and an Excalidraw knowledge map exist together. The source set covered topics such as:

- precious metals and commodities
- social insurance and public-policy explainers
- finance fundamentals for beginners
- investment-product concepts and risk boundaries

These notes intentionally do not list local filenames, non-public folder names, or source document paths. Treat the corpus as a style reference, not as reusable content.

## Excluded Sources

Do not learn default voice from:

- commercial scripts
- book recommendations
- English-learning content
- empty, temporary, or locked document files
- unrelated B-roll source documents

These may be used only when the user explicitly asks to write ads, book recommendations, English learning content, or to edit those exact files.

## Abstracted Style Observations

- The scripts usually open with a strong practical conclusion.
- They often warn the viewer not to confuse a short-term event with a long-term mechanism.
- They move from simple terms to deeper mechanisms rather than starting with theory.
- They use history, policy, formulas, or charts as teaching evidence.
- They repeatedly separate "what people think" from "what actually matters".
- The matching Excalidraw files keep key claims and definitions as short reusable blocks, so scripts should also preserve clean sentence units that can become canvas labels.
